# zhoupeihao's Blog 🌃

A cyberpunk-themed personal blog built with Jekyll.

## About

Welcome to my digital corner of the web — where code, design, and ideas intersect under a neon-lit sky. I'm **zhoupeihao**, a developer passionate about system architecture, DevOps, and pushing boundaries of what's possible.

## Tech Stack

- **Jekyll** — static site generation
- **Custom cyberpunk CSS** — purple/cyan neon aesthetic with animated starfield
- **GitHub Pages** — zero-cost hosting & automatic deployments

## Structure

```
├── _docs/         # In-depth articles and guides
│   └── getting-started.md
├── _posts/        # Blog posts
├── css/           # Cyberpunk theme stylesheets
├── img/           # Generated assets (avatars, favicons)
└── index.html     # Homepage — latest entries
```

## Running Locally

```bash
bundle install
jekyll serve --host 0.0.0.0
# → http://localhost:4000
```
